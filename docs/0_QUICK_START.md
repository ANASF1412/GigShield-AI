# JARVIS EnviroSense Assurance 🚀

### What is it?
An autonomous, zero-touch parametric insurance platform protecting gig workers from extreme weather income loss via real-time sensors, eliminating manual claims.

### Key Innovations
*   **No Claims:** Sensors breach thresholds; the system auto-settles payouts.
*   **Smart Fraud:** Isolation Forest ML + Rule-based Circuit breakers defend liquidity limits.
*   **Cloud/Edge Resiliency:** Supabase persistent logic paired with indestructible memory cache failover.
*   **Explainable AI:** Gemini LLM chat translates complex actuarial decisions strictly via JSON memory logs.

### How to Run (Local)
1. `pip install -r requirements.txt`
2. Configure `.env` and `.streamlit/secrets.toml` 
3. Run `streamlit run app.py`

### Demo Paths
*   **Worker View:** Log in as W001 to observe live environmental risk and coverage stats.
*   **System Defense:** Log as Admin to manually push a simulated event and watch the Circuit Breakers defend the pool limit in real-time.
